#  CHECK-APP

O **Check-App** é uma plataforma inovadora desenvolvida como **plataforma**, com o objetivo de ajudar os usuarios a planejar eventos, festas, reuniões ou tarefas, com praticidade e pouco esforço

## 📌 Funcionalidade do Aplicativo

O Projeto foi idealizado para:

- Facilitar organização.  
- Proporcionar um ambiente,onde o usuario pode controlar as tarefas do evento  
- Exibir eventos futuros .  
- Acompanhar lista de convidados e controlar pessoas ausentes e presentes no evento.  
- Oferecer uma interface prática, simples e intuitiva para qualquer usuário.  

---

## 🎯 Propósito

O Check-App busca:

- Praticidade aos usuarios.  
- Oferecerer gestão de eventos com autonomia e facilidade de uso.  
- Ajudar a quem possa interessar, uma opção de acompanhar eventos e tarefas de forma pratica.  

---

# 🔄 Fluxo de Funcionamento

## 👤 **Usuario**

### **1. Login**
Usuários já cadastrados inserem suas credenciais para acessar o app.  
A opção *"Cadastre-se"* direciona para o registro.

### **2. Cadastro**
O usuário preenche os dados solicitados.  
Após confirmar, já pode fazer login.

### **3. Cria evento**
O usuario cria o evento conforme sua necessidade,seja reunião,eventos,festas ou outros, incluindo local,data e hora.

### **4. Pós-criação**
Após criar o evento, o usuario pode incluir tarefas,marcar tarefas concluidas, ou excluir tarefas, assim como incluir convidados, e acompanhar sua presença ou ausencia.

### **5. Evento concluido**
Após a finalização do evento o usuario,pode estar excluindo o evento para manter a tela limpa para proximos eventos.
